## 题目 1：Unity 中判断两点之间距离有几种方式

常用方式有两种。

第一种：

```csharp
Vector3.Distance(a, b)
```

第二种：

```csharp
(a - b).magnitude
```

理解：

- `a - b` 得到从一个点指向另一个点的向量；
- `.magnitude` 得到这个向量的长度；
- `Vector3.Distance(a, b)` 本质上就是两点差值向量的模长。

## 题目 2：计算向量 `(3, 4, 5)` 的模长

向量模长公式：

```text
|v| = sqrt(x^2 + y^2 + z^2)
```

代入 `(3, 4, 5)`：

```text
sqrt(3^2 + 4^2 + 5^2)
= sqrt(9 + 16 + 25)
= sqrt(50)
= 5 * sqrt(2)
≈ 7.07
```

结论：`(3, 4, 5)` 的模长是 `sqrt(50)`，约等于 `7.07`。

## 题目 3：计算向量 `(3, -4)` 的单位向量

先求模长：

```text
sqrt(3^2 + (-4)^2)
= sqrt(9 + 16)
= sqrt(25)
= 5
```

单位向量公式：

```text
单位向量 = 原向量 / 模长
```

代入：

```text
(3 / 5, -4 / 5)
= (0.6, -0.8)
```

如果放到 Unity 的 `Vector3` 语境里，可以写成：

```text
(0.6, -0.8, 0)
```

## 本次理解

- 距离可以理解成“两点之间差值向量的长度”。
- 模长就是向量每个分量平方相加后再开平方。
- 单位向量就是保留方向，把长度变成 `1`。
- 手算单位向量时，先算模长，再让每个分量除以模长。
