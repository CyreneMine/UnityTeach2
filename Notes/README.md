# Notes

这里保存 UnityTeach2 的课程复盘笔记。

建议按课程或主题创建文件，例如：

- `Lesson01-Overview.md`
- `Mathf-And-Trigonometry.md`
- `Vector-And-Quaternion.md`
- `Resources-And-SceneLoading.md`
- `LineRenderer-And-Raycast.md`
- `Practice-Project-Review.md`

每篇笔记建议记录：

- 对应视频；
- 核心知识；
- 自己一开始理解错的点；
- 代码、场景或 Inspector 中遇到的问题；
- 为什么会出错；
- 如何验证；
- 尚未验证的边界；
- 下一步。

不要把未经验证的结论写成确定结论。后续发现理解有误时，追加更正说明。
